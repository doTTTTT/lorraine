# Lorraine

Lorraine is a work management framework for tasks.

### Get started

First add the dependency to your project:

```toml
[versions]
lorraine = "0.0.1"

[libraries]
lorraine = { module = "io.github.dottttt.lorraine:lorraine", version.ref = "lorraine" }
```
